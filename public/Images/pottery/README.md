# Pottery photos

Drop your pot photos in this folder with these exact names and they'll show inside
the field notes on `/pottery` (the shelf keeps the little drawings; the photo only
appears when you click a piece):

- `celadon-bowl.jpg` — the green bowl (the first one you made)
- `honey-cup.jpg` — the amber / mustard honey cup
- `speckle-bowl.jpg` — the blue speckled bowl

Any orientation is fine — the field-notes plate crops to a square, so a roughly
centred pot works best. If you use different filenames, update the `photo` paths in
`src/Utils/potteryData.js`. Until the files are here, each piece falls back to its
drawing (no broken images).
